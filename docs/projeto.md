# Agenda de Marcar consultas  

## Descrição do Projeto  
    - Um projeto em python para marcação de consultas
        - Componentes
            - Capacidade de criação de consultas
            - Lógica para não marcação de consultas em cima da outra
            - Sistema de log
            - CRUD de usuários via API
        - Regras
            - Consultas são para apenas 1 usuário.
            - Não se pode ter mais que uma consulta com o mesmo especialista ao mesmo tempo
            - Precisa ter especialistas, usuários e locais cadastrados para poder marcar consultas
